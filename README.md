# Percept
[Installation YouTube Tutorial](https://www.youtube.com/watch?v=YGhYX8cglvg&pp=2AYX)

AssistantExtensions plugin that replaces all Siri queries to ChatGPT.

Compatible with iOS 5.1+

You can add https://calvinkNT.github.io/Percept/repo as a repository in Cydia to get the latest version automatically.
